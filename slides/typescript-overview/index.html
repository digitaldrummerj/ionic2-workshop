What is TypeScript?

  • Typed Superset of JavaScript
  • Compiles to plain JavaScript
  • Works with Any Browser, Host, or OS
  • Open Source
  • Lots of Tooling Support
  • .ts file extension

Key Features

  • Static Typing
  • Design TIme Data Type Checks
  • Classes and Modules
  • Interfaces
  • Arrow functions
  • Supports Standard JavaScript

Installing

npm install -g typescript

Syntax Rules

Types

  • Everything derives from object
  • number
  • string
  • boolean
  • Array

Types, cont'd

  • "any" type
  • Anything goes


function sayHi(sayer: any) {
	console.log("Hi from " + sayer);
}

sayHi("Justin");
sayHi("11");

Declaring Strings

Can use double or single quotes

let hisName = "George";
let herName = 'Tracy';

Multiline Strings

Just use backticks

let name = `
Hi from ${hisname}
and ${herName}
`;

Variables

Largely the same as JavaScript

var name = "Justin";

var name: string = "Justin";

Type Checking

Warnings when types not the same

var name = "Justin";

name = 3;
Type 'number' is not assignable to type 'string'

Variable Strangeness

Did you notice that we declare the same variable twice?

var name = "Justin";

var name: string = "Justin";

This is totally valid

Instead

use let or const

let name = "Justin";

let name: string = "Justin";
Cannot redeclare block-scoped variable 'name'

Classes

 

Defines Behaviors and Properties

Defining a Class


class Animal {
 name: string,
 constructor(theName: string) {
      this.name = theName;
 }
 move(distanceInMeters: number = 0){
    console.log(`${this.name} moved ${distanceInMeters}m.`);
 }
}
Class Name Properties Constructor Function with Parameter

Inheritance

class Snake extends Animal {
  constructor(name: string) {
     super(name);
  }

  move(distanceInMeters = 5) {
 	super.move(distanceInMeters);
  }
}
Base Class Call Base Class Constructor Call Base Class Move method

Using Classes


let sam = new Snake("Sammy the Python");
sam.move();

let tom: Animal = new Animal("Tommy the Palomino");
tom.move(34);

Constructor Property Shorthand


class Animal {
 constructor(public name: string) { 
 }
 
 move(distanceInMeters: number = 0){
    console.log(`${this.name} moved ${distanceInMeters}m.`);
 }
}
